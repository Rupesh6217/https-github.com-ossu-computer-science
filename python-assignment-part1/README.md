# Student Grade Tracker

This project is a simple command-line program built using Python. It helps in managing student records, calculating grades, and generating basic performance reports.

## Features
- Cleans and organizes raw student data  
- Calculates grades based on marks  
- Shows overall class performance  
- Performs basic text processing tasks  

## Technologies Used
- Python  
- Loops and conditional statements  
- String operations
